# journeypedia

A chrome extension to record your wikipedia journeys.

## Game
The game involves finding a path from Article A to Article B.

An example of this is: Cat -> Neutering -> Canine reproduction -> Canis -> Dog.

The extension records the time you took to find the path and the steps you took to get there. It also highlights links in the article for better visibility.

## Rules
- No using the Find shortut (Ctrl/Cmd + F).
- You can open new WikiPedia windows so long as the URL exists on the previous article.
